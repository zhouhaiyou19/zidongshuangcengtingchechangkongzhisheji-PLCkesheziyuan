                              ## 课程设计说明

                              设计的规则如下：
                              1. 1、2、3车位只能上下移动，不能左右移动。
                              2. 4、5车位只可左右移动，不可上下移动。
                              3. 下排车位只需直接将车子开出即可。
                              4. 如要呼叫上排车子，只需按下1至3的按钮，再按下“叫车”按钮，则所按车位将降至下层，而下排车位将左右移动，让出位置让上层车位降下。
